Access web

El objetivo de esta web es brindar recursos y recomendaciones para que los desarrolladores de productos digitales puedan generar contenido accesible para todas las personas.

h2> Extensiones

Extensión AXE debtools

verificador de Accesibilidad para Developers, Testers, y Designers en Chrome. encuentre y solucione mas problemas de accesibilidad durante el desarrollo del sitio web con axe DevTools.

Extensión Wave Evaluation Tool

WAVE es una herramienta de evaluación de accesibilidad web desarrollada por WebAIM.org. Proporciona información visual sobre la accesibilidad de su contenido web mediante la inyección de iconos e indicadores en su página. Ninguna herramienta automatizada puede decirle si su página es accesible, pero WAVE facilita la evaluación humana y educa sobre problemas de accesibilidad. Todo el análisis se realiza completamente dentro del navegador Chrome, lo que permite una valoración segura de la intranet, local, protegida con contraseña y otras páginas web confidenciales.

descargar Extensión WAVE para Chrome en Google Web Store

Extensión Lighthouse

Lighthouse es una herramienta automatizada de código abierto para mejorar el rendimiento, la calidad y la corrección de sus aplicaciones web. Al auditar una página, Lighthouse ejecuta un aluvión de pruebas contra la página y luego genera un informe sobre qué tan bien lo hizo la página. Desde aquí puede utilizar las pruebas de falla como indicadores de lo que puede hacer para mejorar su aplicación.
